Markdown语法简介

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面。
—— 维基百科

例如本文，使用简单的符号可以标识不同的标题，将某些文字标记为粗体， 斜体， 删除线，还可以插入代码puts "helloworld"，更高级地还可以插入表格、代码块和公式，例如：

代码块

#include <stdio.h>

int main() {
   int a = 23, b = 47;
   
   printf("Before. a: %d, b: %d\n", a, b);
   a ^= b;
   b ^= a;
   a ^= b;
   printf("After.  a: %d, b: %d\n", a, b);
   return 0;
}
